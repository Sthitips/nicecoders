# Circular Fashion AI
## Database Schema

Database: PostgreSQL

---

# Users

Stores authentication and profile information.

| Column | Type | Constraints |
|---------|------|-------------|
| id | UUID | Primary Key |
| username | VARCHAR(50) | Unique |
| email | VARCHAR(100) | Unique |
| password_hash | TEXT | Not Null |
| full_name | VARCHAR(100) | |
| avatar_url | TEXT | Nullable |
| bio | TEXT | Nullable |
| location | VARCHAR(100) | Nullable |
| created_at | TIMESTAMP | Default NOW() |

Relationships
- Owns many wardrobe items
- Owns many listings
- Has one Style DNA
- Has one Sustainability record
- Can participate in many chats

---

# Wardrobe

Stores clothing uploaded by users.

| Column | Type |
|---------|------|
| id | UUID |
| owner_id | UUID |
| image_url | TEXT |
| category | VARCHAR(50) |
| color | VARCHAR(50) |
| material | VARCHAR(50) |
| brand | VARCHAR(50) |
| style | VARCHAR(50) |
| season | VARCHAR(30) |
| occasion | VARCHAR(50) |
| ai_description | TEXT |
| embedding | VECTOR (optional initially) |
| created_at | TIMESTAMP |

Relationships

Belongs to one User

Can become one Marketplace Listing

---

# Listings

Marketplace listings.

| Column | Type |
|---------|------|
| id | UUID |
| wardrobe_item_id | UUID |
| owner_id | UUID |
| title | VARCHAR(150) |
| description | TEXT |
| listing_type | ENUM(BUY, RENT, SWAP) |
| selling_price | DECIMAL |
| rent_price | DECIMAL |
| status | ENUM(ACTIVE,SOLD,RENTED) |
| ai_generated | BOOLEAN |
| created_at | TIMESTAMP |

# Style DNA

| Column | Type |
|---------|------|
| id | UUID |
| user_id | UUID |
| favourite_styles | JSON |
| favourite_colors | JSON |
| favourite_categories | JSON |
| embedding | VECTOR (optional) |
| last_updated | TIMESTAMP |

Example

Streetwear

Minimal

Dark Academia

Vintage

# AI Recommendations

Stores recommendations generated by AI.

| Column | Type |
|---------|------|
| id | UUID |
| user_id | UUID |
| listing_id | UUID |
| recommendation_score | FLOAT |

Example

"Frequently liked similar colors"
